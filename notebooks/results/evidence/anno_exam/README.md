# Annotation Evidence

This folder contains 3–5 representative examples of annotated training images from the Roboflow dataset (YOLOv8 format).

Each image demonstrates:

- Correct bounding box placement
- Proper class labeling (`hard_hat`, `no_hard_hat`)
- Coverage of typical construction site scenarios
- Inclusion of at least one challenging case (e.g., occlusion or small object)

## Purpose

These screenshots provide visual confirmation that:

- The dataset is correctly labeled
- Bounding boxes are aligned with target objects
- The supervised learning pipeline is based on valid ground truth annotations

Annotation quality directly impacts model performance and error behavior (false positives and false negatives).
